# Lista de validación del rediseño

## Coherencia académica

- [ ] El nombre definitivo de la materia indica Álgebra Lineal.
- [ ] Las competencias y objetivos se corresponden con Álgebra Lineal, no con metodología de investigación.
- [ ] Las sesiones 1 y 2 fueron sustituidas y revisadas por la autoridad académica.
- [ ] Las 28 sesiones mantienen continuidad conceptual.
- [ ] La bibliografía usada corresponde a los contenidos de cada bloque.

## Alineación didáctica

- [ ] Cada sesión tiene preparación previa, actividad sincrónica y consolidación posterior.
- [ ] La preparación previa es breve y necesaria para el trabajo de la sesión.
- [ ] La actividad sincrónica exige análisis, aplicación, justificación o comparación.
- [ ] Cada resultado de aprendizaje utiliza un verbo observable.
- [ ] La evidencia solicitada permite verificar el resultado de aprendizaje.
- [ ] Las tareas posteriores no duplican innecesariamente el trabajo sincrónico.

## Modalidad híbrida

- [ ] Cada actividad puede realizarse presencialmente o en línea.
- [ ] Se definieron plataforma, canal de comunicación y mecanismo de entrega autorizados.
- [ ] Se prevé cómo participarán estudiantes conectados a distancia en equipos mixtos.
- [ ] Las reglas de uso de calculadora, software e IA están explicitadas.

## Evaluación

- [ ] Se conserva la ponderación global: 30 % primer parcial, 30 % segundo parcial y 40 % evaluación final.
- [ ] El primer parcial cubre predominantemente el bloque de matrices, determinantes y sistemas.
- [ ] El segundo parcial cubre predominantemente espacios vectoriales y transformaciones lineales.
- [ ] La evaluación final recoge valores propios, aproximación y descomposiciones, según la programación oficial.
- [ ] Los instrumentos evalúan procedimientos, justificaciones y resultados, no solo respuestas finales.
- [ ] Las fechas y modalidades de evaluación fueron confirmadas institucionalmente.

## Transparencia de fuentes

- [ ] Los contenidos tomados del sílabo se distinguen de las recomendaciones pedagógicas generadas.
- [ ] El PDF de Bases de Datos se usa solo como referencia de estructura didáctica.
- [ ] El banco de preguntas de metodología se conserva solo como ejemplo de formato/práctica, no como evaluación de Álgebra Lineal.
