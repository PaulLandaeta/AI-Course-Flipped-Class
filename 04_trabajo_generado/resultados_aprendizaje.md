# Tres versiones de resultados de aprendizaje

## Unidad seleccionada

**Matrices, determinantes y sistemas lineales** (sesiones 1–11 del plan rediseñado).

**Nota de alcance:** el sílabo base nombra determinantes, sistemas lineales, Gauss-Jordan, descomposición LU y matriz inversa. Las formulaciones, condiciones, evidencias y justificaciones siguientes son propuestas pedagógicas generadas para hacer evaluables esos contenidos.

## Versión 1: resultado integrador orientado a resolución

> **Resuelve** sistemas lineales y problemas de aplicación mediante representación matricial, determinantes, eliminación, Gauss-Jordan, regla de Cramer, matriz inversa o descomposición LU, **seleccionando y justificando** el método más pertinente en ejercicios individuales y colaborativos de complejidad progresiva.

| Elemento | Especificación |
|---|---|
| Verbo observable | Resuelve; selecciona; justifica. |
| Contenido o conocimiento | Sistemas lineales, matrices, determinantes, regla de Cramer, eliminación, Gauss-Jordan, matriz inversa y LU. |
| Condición de aplicación | Ante ejercicios y problemas de aplicación de complejidad progresiva, en modalidad híbrida, con trabajo individual y colaborativo. |
| Nivel de Bloom | Aplicar y analizar. |
| Evidencia verificable | Problema resuelto con procedimiento completo, verificación de la solución y una justificación breve de la elección del método. |
| Por qué es medible | Se puede comprobar si el estudiante formula el sistema correctamente, ejecuta el procedimiento sin errores críticos, obtiene/verifica la solución y sustenta una elección de método con criterios matemáticos. |

## Versión 2: resultado orientado a análisis y decisión de método

> **Analiza** la estructura de una matriz o sistema lineal para **clasificar** sus posibles soluciones y **determinar** si corresponde utilizar determinantes, reducción por filas, matriz inversa o una descomposición LU, **a partir de datos simbólicos y problemas contextualizados**.

| Elemento | Especificación |
|---|---|
| Verbo observable | Analiza; clasifica; determina. |
| Contenido o conocimiento | Determinantes, pivotes, sistemas compatibles e incompatibles, sistemas determinados, subdeterminados y sobredeterminados; métodos de solución. |
| Condición de aplicación | A partir de matrices, sistemas simbólicos y problemas contextualizados entregados por el docente. |
| Nivel de Bloom | Analizar. |
| Evidencia verificable | Cuadro de clasificación de casos, elección de método para cada caso y explicación escrita o oral de los criterios empleados. |
| Por qué es medible | La clasificación admite criterios explícitos; la decisión puede contrastarse con la estructura del sistema y la argumentación permite verificar que no fue una elección aleatoria. |

## Versión 3: resultado orientado a modelación y validación

> **Modela y valida** situaciones de aplicación mediante sistemas de ecuaciones lineales, **traduciendo** las variables y restricciones a una representación matricial, **resolviendo** el modelo y **evaluando** la coherencia de la solución con las condiciones del problema.

| Elemento | Especificación |
|---|---|
| Verbo observable | Modela; traduce; resuelve; evalúa. |
| Contenido o conocimiento | Variables, ecuaciones lineales, representación matricial, resolución de sistemas y comprobación de soluciones. |
| Condición de aplicación | Frente a un problema de aplicación proporcionado durante una actividad híbrida o evaluación individual. |
| Nivel de Bloom | Aplicar, analizar y evaluar. |
| Evidencia verificable | Entrega de un modelo con definición de variables, sistema/matriz, desarrollo de la solución, sustitución o verificación, e interpretación final en el contexto planteado. |
| Por qué es medible | Cada parte del desempeño es observable y calificable con criterios: correspondencia entre datos y ecuaciones, corrección del cálculo, verificación y coherencia de la interpretación. |

## Recomendación de uso

La **versión 1** puede funcionar como resultado principal de la unidad porque integra cálculo, selección de procedimiento y justificación. Las versiones 2 y 3 pueden emplearse como resultados específicos o criterios de evaluación asociados.
